工具箱

详细介绍：

  • 账号本子：

    • 功能描述：记录账号密码信息（包含生成随机密码功能），并加密保存于数据库，进入应用有应用锁（支持指纹、手势和PIN密码）
    • 阶段：实现基础功能，加密处于开发阶段。

  • 一本日记:

    • 功能描述：记录日记，支持markdown语法，以时间线方式展示
    • 阶段：未开发

  • 光能使者：

    • 功能描述：光能使者召唤符文动画
    • 阶段：完成

  • 壁纸库：

    • 功能描述：将Wall Haven网站的壁纸资源整合到手机中，目前有3个标签，分别为随机、最新和热榜，可下载
    • 阶段：功能完成，性能有待优化

  • 移动的小球：

    • 功能描述：手指滑动可给小球提供一个初速度，滑动加速度可调，重力可调。
    • 阶段：完成

  • 连连看：

    • 功能描述：就是个连连看。
    • 阶段：两折点连接未实现

  • CropperImageView测试：

    • 功能描述：smart cropper库的使用测试，拍照，识别照片中的矩形物体，可进行透视裁剪。适用于身份证，名片，文档等照片的裁剪。
    • 阶段：完成
